## Функциональность

### Форма поиска

Четыре поля для поиска (все необязательные):
- Имя пользователя (username) — поиск в /users
- Email пользователя — поиск в /users
- Заголовок поста — поиск в /posts
- Текст поста — поиск в /posts

### Что происходит при поиске

1. Проверка заполненных полей
2. Параллельные запросы к API через asyncio.gather
3. Фильтрация данных на стороне клиента
4. Параллельная загрузка имён авторов для постов
5. Отображение результатов в двух блоках

### Отображение результатов

**Пользователи:** name, username, email, phone, website

**Посты:** title, body (первые 150 символов), userId, имя автора

## Ключевые асинхронные конструкции

| Конструкция | Назначение |
|-------------|------------|
| `async def` | Объявление асинхронной функции |
| `await` | Ожидание результата асинхронной операции |
| `asyncio.gather()` | Параллельное выполнение нескольких задач |
| `asyncio.run()` | Запуск асинхронной функции из синхронного кода |
| `async with` | Асинхронный контекстный менеджер |

### Пример параллельных запросов

```python
users, posts = await asyncio.gather(
    get_users(session),
    get_posts(session)
)
```
